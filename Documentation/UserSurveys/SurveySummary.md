# User Survey Evidence

## Description
This folder contains screenshots of conversations with customers and professionals regarding the problem of bias in marketing content.

## Purpose
The purpose of these interactions was to validate:
- Whether AI-generated marketing content contains bias
- Whether professionals face this issue
- Whether there is a need for an automated solution

## Summary of Responses
- Customers confirmed that AI tools are widely used
- Some professionals observed bias in generated content
- Manual review is commonly used but not always effective
- Many showed interest in a tool that can detect ethical issues

## Conclusion
These interactions validate that the problem exists and there is a real need for a solution like AdEthix AI.
